**Example 2: Dashboard Generation (Python Script)**
```python
# Auto-generated dashboard for sales_data.xlsx (using Plotly)
import plotly.express as px
import pandas as pd

# Load data (Agent would pre-process this)
df = pd.read_excel("sales_data.xlsx", sheet_name="Q1_2024")

# Revenue by Product (Bar Chart)
fig1 = px.bar(
    df,
    x="Product_Name",
    y="Revenue",
    title="Revenue by Product (Q1 2024)",
    color="Region"
)
fig1.show()

# Monthly Trends (Line Chart)
df["Month"] = df["Date"].dt.to_period("M")
monthly_revenue = df.groupby("Month")["Revenue"].sum().reset_index()
fig2 = px.line(
    monthly_revenue,
    x="Month",
    y="Revenue",
    title="Monthly Revenue Trend",
    markers=True
)
fig2.show()
```
